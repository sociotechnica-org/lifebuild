# Job 6: Playbook Drafting

Research and draft a communications playbook as source material for Sam.

## Trigger

Human asks Arvin to create a playbook for a specific communications scenario (inbound triage, content briefing, group event, relationship follow-up, etc.).

## Procedure

1. Read the relevant division card(s) and inventory entry
2. Read the acquihire brief to understand strategic context
3. Research best practices for this specific scenario via web search
4. Draft the playbook as a source document
5. Include: when to use it, step-by-step procedure, decision points, examples, anti-examples

## Output

A source document at `docs/context-library/sources/playbook-[name]-[date].md` that Sam can turn into a Playbook card.
